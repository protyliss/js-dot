Class ObjectMap<TObject, TMap>

Object Mapper for Renamed Properties

Example

Default Usage
const map = new ObjectMap({
 alpha: 'foo',
 bravo: 'bar.baz'
});

const item = {foo: 1, bar: {baz: 2}}

// return 1
map.get(item, 'alpha');

// return {alpha: 1, bravo: 2}
map.getAll(item);
