What's a Dot?

The concepts of "Dot, Line and Plane" are inspired by design principles.

A Dot represents base-level libraries specific to a programming language. "Dot to Dot" creates a Line such as relationship between JavaScript and HTML. "Line to Line" creates a Plane, such as Website based on Javascript, HTML, CSS and Server-side language.
